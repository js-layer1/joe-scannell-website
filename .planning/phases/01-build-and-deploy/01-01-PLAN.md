  <name>Task 1: Bootstrap Next.js project and configure static export</name>
  <files>package.json, next.config.ts, tsconfig.json, .gitignore, app/globals.css, app/layout.tsx</files>
  <action>
    The working directory is `/Users/joe/Claude Code/Projects/joe-scannell-website`. The repository already exists. Bootstrap the Next.js 16.1.6 project into this directory using:

    ```bash
    npx create-next-app@16.1.6 . --typescript --eslint --app --tailwind --yes
    ```

    After scaffolding:

    1. Replace `next.config.ts` with the static export configuration (CRITICAL -- both settings required):
    ```typescript
    import type { NextConfig } from 'next'
    const nextConfig: NextConfig = {
      output: 'export',
      images: {
        unoptimized: true,
      },
    }
    export default nextConfig
    ```
    Without `images.unoptimized: true`, `next build` fails with a static export error.

    2. Replace `app/globals.css` with Tailwind v4 CSS-first config:
    ```css
    @import "tailwindcss";

    :root {
      --font-geist: '';
    }

    body {
      font-family: var(--font-geist), system-ui, sans-serif;
    }
    ```

    3. Replace `app/layout.tsx` with a minimal shell (NO metadata export yet -- that goes in Plan 01-03). Use Geist font via `next/font/google`:
    ```typescript
    import type { ReactNode } from 'react'
    import { Geist } from 'next/font/google'
    import './globals.css'

    const geist = Geist({
      subsets: ['latin'],
      variable: '--font-geist',
      display: 'swap',
    })

    export default function RootLayout({ children }: { children: ReactNode }) {
      return (
        <html lang="en" className={geist.variable}>
          <body className="antialiased">
            {children}
          </body>
        </html>
      )
    }
    ```

    4. Install schema-dts (needed for Plan 01-03 but install now):
    ```bash
    npm install schema-dts
    ```

    5. Run `npm run build` and fix any errors before proceeding. Common issue: if create-next-app generates conflicting Tailwind config, remove `tailwind.config.ts` if it exists (Tailwind v4 uses CSS-first config, not a JS config file).

    Font choice rationale: Geist is already hosted on Vercel's CDN, warms up instantly, and feels contemporary without being cold/technical. It suits the warm personal aesthetic.
  </action>

  </verify>
  <done>

  </done>
